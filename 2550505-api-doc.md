## 一、基础信息

- **基础域名**：`https://2550505.com`  
- **返回格式**：主要为 `application/json`（除首页 HTML、静态资源等）
- **认证方式**：
  - 普通用户态接口通过 Cookie / 会话识别当前登录用户；
  - 登录相关接口在请求头中携带 `hash` 字段，Body 为 JSON。
- **请求错误**：
  ```json 
    {
        code: xxx, //错误码 成功时为 0
        msg: "" //错误消息
    }
  ```
